| 2026-04-22 | Weather-aware heating owns the Wiser Kitchen/LR/Office setpoint at 22:00 daily, base+offset absolute (not delta) | Research showed two paradigms: absolute (system overwrites whatever's currently set with base+offset) vs delta (apply offset to whatever the user most recently set). | 2026-04-27 | **Trigger source for sit-still-friendly motion lights: the Zigbee device's `_occupancy` entity, OR'd with the `_motion` entity into a presence group.** Don't trigger only on PIR motion. | Edgar's office sensors are 4-in-2: each physical sensor exposes both a `motion` (binary, fires on movement, clears in seconds) and an `occupancy` (binary, holds `on` for as long as a person is in range — typically driven by mmWave on the same chip). The group goes `off` only when ALL four inputs report off — which means no movement *and* no presence.
